# Isro-Journey
# 🚀 ISRO Journey - India's Space Exploration Timeline

![ISRO Journey](https://img.shields.io/badge/Status-Live-success)
![Version](https://img.shields.io/badge/Version-1.0-blue)

> A modern, interactive web application showcasing India's remarkable journey in space exploration from 1975 to 2023 and beyond.

## 🌟 Live Demo

👉 **[Visit Website](https://sanidhyasharma137.github.io/isro-journey)**

## ✨ Features

- 📅 **Interactive Horizontal Timeline** - Scroll through ISRO's major milestones
- 🔍 **Smart Search** - Find missions by name, type, or description
- 🎯 **Category Filters** - Filter by Satellites, Lunar, Mars, and Rockets
- 🌓 **Dark/Light Mode** - Toggle themes for comfortable viewing
- ⌨️ **Keyboard Shortcuts** - D (Theme), S (Search), T (Top), Esc (Close)
- 📤 **Share Button** - Share on WhatsApp, Twitter, LinkedIn
- 📊 **Visual Stats Chart** - Mission success visualization
- 📱 **Fully Responsive** - Works on all devices
- 🎨 **Smooth Animations** - Optimized for performance

## 🛰️ Missions Covered

| Year | Mission | Achievement |
|------|---------|-------------|
| 1975 | Aryabhata | India's first satellite |
| 1980 | SLV-3 | First Indian launch vehicle |
| 1982 | INSAT-1A | First communication satellite |
| 2008 | Chandrayaan-1 | Discovered water on Moon! 💧 |
| 2013 | Mangalyaan | Mars mission - First attempt success! 🔴 |
| 2019 | Chandrayaan-2 | Moon landing attempt |
| 2023 | Chandrayaan-3 | Successful Moon landing! 🎉 |
